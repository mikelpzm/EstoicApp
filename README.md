# Meditaciones - Marco Aurelio

Una aplicación web elegante para explorar las meditaciones del emperador filósofo romano Marco Aurelio (121-180 d.C.).

## Características

- **Meditación del día**: Cada día muestra una meditación diferente para reflexionar
- **Explorar por temáticas**: Filtra las 80 meditaciones por 10 categorías temáticas:
  - Virtud
  - Muerte y Mortalidad
  - Naturaleza
  - Deber y Responsabilidad
  - Control Mental
  - Tiempo y Transitoriedad
  - Adversidad
  - Relaciones Humanas
  - Simplicidad
  - Sabiduría
- **Compartir**: Comparte fácilmente las meditaciones
- **Diseño responsive**: Funciona en móvil y escritorio
- **Tema oscuro**: Diseño elegante con acentos dorados

## Tecnologías

- React 19
- Vite
- CSS moderno (variables, grid, flexbox)
- Google Fonts (Cormorant Garamond, Inter)

## Instalación

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Sobre las Meditaciones

"Meditaciones" es una serie de escritos personales del emperador romano Marco Aurelio, registrando sus ideas sobre la filosofía estoica. Originalmente escritas en griego antiguo entre el 170 y 180 d.C., la obra ha sido una fuente de sabiduría e inspiración durante casi dos milenios.
